Background Images

To use a background image on an element:

  • background-image: url(path.jpg);

To repeat a background image:

  • background-image: repeat; (to tile it)
  • background-image: repeat-x; (tile horizontally)
  • background-image: repeat-y; (tile vertically)
  • background-image: no-repeat; (no tiling)

To attach a background:

  • background-attachment: fixed;
  • background-attachment: scroll; (default value)

To position a background image:

  • background-position: x y;
  • Where x and y can be percentages or absolute values
  • x can be "left" "center" "right"
  • y can be "top" "center" "bottom"

Using background shorthand:

  • Can place all background properties on one line using the property background. For example:
  • background: url(path.jpg) repeat fixed center top;

*Tip: It's good practice to use a background-color as a fallback.

  • background-color: ##add8e6; /*fallback*/
  • background: url(path.jpg) no-repeat center;

Background Size:

  • background-size: width height;
  • Width and height are px values or percentages. If you only give it a width, the height is set to auto.
  • background-size: cover;
  • Scales the background to cover the entire area, might cut off some of the background to make it fit.
  • background-size: contain;
  • Scales the image to it's largest size so that both width and height can fit the container. Doesn't cut off any of the image.
  • To use within shorthand, the value/s must come after the position values and it's separated by /
