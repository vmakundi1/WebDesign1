In-class Assessment Four

Welcome to Assessment Four

In this assessment you will build a web page that:

  • Utilizes a footer and a header that use one hundred percent of the width of the browser.
  • The footer will “stick” to the bottom of the viewport.
  • The main content text will be center aligned in the center of the browser.
  • All the text will scale relatively to the user settings of the browser.
  • The colors, graphics, images and content text are all provided in this wireframe.

Relative Font Sizing

In this assessment you will build a webpage that uses a relatively sized font. The font-size property specifies the size, or height, of the font. Font-size affects not only the font to which it is applied, but is also used to compute the value of em, rem, and ex length units. Relative fonts are one part of the responsiveness of the page.

CSS Sticky Footer

Another aspect of this page is the sticky footer; the purpose of a sticky footer is that it "sticks" to the bottom of the browser window. But not always, if there is enough content on the page to push the footer lower, it still does that. But if the content on the page is short, a sticky footer will still hang to the bottom of the browser window. This is great if you have a full-page background that you want the user to see but not enough content to fill the viewport.

Liquid Containers

The third and final component of this web page is the liquid containers. The term "liquid" implies that a Website should flow smoothly into whatever space it is given. If you use a high-resolution monitor, this may mean that you need to resize your browser a little, which most people in that situation do. If you have a low-resolution monitor, you will still see the information, it will just be a little more compact. Liquid design is another part of the responsiveness of the page.
